# Code of Conduct

This is a small educational project; the rules are correspondingly simple.

- Be respectful and constructive in issues, discussions and pull requests.
- Critique code and ideas, not people. Assume good intent.
- No harassment, personal attacks, or discriminatory language of any kind.
- Keep discussions on topic and give newcomers room to ask basic questions.

Maintainers may edit, hide or remove contributions that violate these rules,
and may block repeat offenders. To report a problem, open an issue on the
repository (or use GitHub's report feature for content you'd rather not
discuss publicly).
